# 🍽️ Zomato Restaurant Insights

## 📌 Project Overview

This project analyzes the Zomato restaurant dataset to uncover meaningful insights about restaurant ratings, cuisines, pricing patterns, customer preferences, and location-based trends. The analysis was performed using Python and various data analytics libraries to transform raw data into actionable business insights.

## 🎯 Objectives

* Clean and preprocess the Zomato dataset.
* Analyze restaurant ratings and customer preferences.
* Explore relationships between cuisines and ratings.
* Identify restaurant hotspots based on location.
* Examine the impact of pricing on ratings.
* Create visualizations to support data-driven insights.
* Provide business recommendations for restaurant and food-tech platforms.

## 🛠️ Technologies Used

* Python
* Google Colab / Jupyter Notebook
* Pandas
* NumPy
* Matplotlib

* WordCloud

## 📊 Analysis Performed

### Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Cleaned text-based columns.
* Standardized dataset structure.

### Exploratory Data Analysis (EDA)

* Top-rated restaurants analysis.
* Cuisine-wise rating analysis.
* Location hotspot identification.
* Price vs Rating relationship analysis.
* Rating distribution analysis.
* Correlation analysis using heatmaps.

### Visualizations

* Bar Charts
* Scatter Plots
* Heatmaps
* Histograms
* Word Clouds

## 🔍 Key Findings

* Certain cuisines consistently achieve higher customer ratings.
* Popular cities contain a higher concentration of restaurants.
* Restaurant pricing does not always correlate with higher ratings.
* Customer votes strongly influence rating reliability.
* Online delivery services contribute to customer engagement.
